- Handbækur og stuðningslengdir
- Viðhalds- og bilanagreiningarleiðbeiningar
- Tengitöflur

### [QUICK_REFERENCE.md](./QUICK_REFERENCE.md)
Flýtiyfirlit fyrir skjóta uppslátt á mikilvægum upplýsingum:

- Helstu tengingar og stillingar
- Tengiskema
- Stuðningslengdir

### [AVR435_CONTROLS_AND_CONNECTIONS.md](./AVR435_CONTROLS_AND_CONNECTIONS.md) *(Enska)*
Ítarleg handbók fyrir stjórntæki og tengingar Harman Kardon AVR435/230:
- Ítarleg kortlagning og lýsingar á stjórntækjum á framhlið
- Ítarleg kortlagning tenginga á bakhlið (með númeruðum tilvísunum í skýringarmynd)
- Litakóðar fyrir tengingar (CEA staðall)
- Leiðbeiningar um uppsetningu og stillingar
- Leiðbeiningar fyrir bilanaleit
- Flýti-skýringarmyndir

### [PC_TO_AVR_CONNECTION_GUIDE.md](./PC_TO_AVR_CONNECTION_GUIDE.md) *(Enska)*
Ítarlegar leiðbeiningar til að tengja þessa tölvu við AVR435/230 magnarann:
- Greining á öllum tiltækum hljóðútgöngum tölvunnar
- 5 tengimöguleikar með kostum og göllum
- **MÆLT MEÐ: XGIMI Optical lausnin** (best, ódýrust, einföldust)
- Verðsamanburður og meðmæli
- Stillingar fyrir hljóð í Linux (PipeWire)
- Uppsetning fyrir umgerðarhljóð (Surround sound)
- Skriftur fyrir hljóðbeiningu (Audio routing)
- Bilanaleit

### [PC_AUDIO_QUICK_SETUP.md](./PC_AUDIO_QUICK_SETUP.md) *(Enska)*
5 mínútna flýtileiðbeiningar fyrir hljóð frá tölvu í gegnum XGIMI Optical:
- **Tenging**: Tölva → HDMI → XGIMI → Optical → AVR435/230
- Skref-fyrir-skref uppsetning (5 skref, 5 mínútur)
- Innkaupalisti (10$ optical kapall)
- Hljóðstyrksstillingar og bilanaleit
- Tilvísun í Linux skipanir
- Tengiskýringarmynd

### [AVR435_PC_REMOTE_CONTROL.md](./AVR435_PC_REMOTE_CONTROL.md) *(Enska)*
Ítarleg handbók til að fjarstýra AVR435/230 frá tölvu með RS-232:
- **Aðferð**: USB-í-RS232 raðtengisstýring (tvíátta)
- Vélbúnaðarkröfur (~$20-40)
- Tæknilýsingar fyrir RS-232 samskiptareglur
- Python stýriskrifta (fylgir með)
- Tiltækar skipanir (straumur, hljóðstyrkur, inntak, umgerðarhljóð)
- Stillingar og bilanaleit í Linux
- Flýtilyklar og sjálfvirkni

### [AVR_RS232_SHOPPING_LIST.md](./AVR_RS232_SHOPPING_LIST.md) *(Enska)*
Ítarlegur innkaupalisti fyrir RS-232 vélbúnað:
- **Besti kostur**: CableCreation USB-í-RS232 FTDI (~$20)
- Ítarlegur samanburður á vörum með hlekkjum
- Hvað á að kaupa og hvað á að forðast
- Gátlisti til að staðfesta fyrir kaup
- Leiðbeiningar um uppsetningu eftir kaup

### [ICELANDIC_SHOPPING_GUIDE.md](./ICELANDIC_SHOPPING_GUIDE.md) *(Íslenska)*
Ítarlegur íslenskur kauplisti fyrir AVR435 PC tengingar:
- **Optical kaplar**: ELKO.is (2.195-3.995 kr)
- **USB-to-RS232**: Computer.is (2.990 kr) og Amazon samanburður
- Nákvæmar verðupplýsingar frá íslenskum vöruverslunum
- Lengdarmælingar og uppsetningarleiðbeiningar
- Heildarsamanburður: Ísland vs. Amazon
- Verslanaupplýsingar og opnunartímar

Þegar nýjum tækjum er bætt við eða breytingar eru gerðar:

1. Taktu ljósmyndir af tækinu (fram- og aftanverð)
2. Auðkenndu líkan og framleiðanda

4. Uppfærðu `TECH_ROOM_EQUIPMENT.md` með nýjum upplýsingum
5. Uppfærðu `QUICK_REFERENCE.md` ef nauðsynlegt
